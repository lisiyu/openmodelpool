🔗 获得固定公网地址
绑定域名或 IP 后可获得固定地址，方便分享给他人使用。
💡 绑定域名：适合有自己域名的用户，通过 CNAME 解析获得固定 HTTPS 地址
💡 绑定 IP：适合有固定公网 IP 的用户，直接绑定无需域名。如果服务器没有固定 IP，推荐用绑定域名更简单
📋 域名绑定分步指引：
1️⃣ 准备一个域名（推荐在 Cloudflare / 阿里云 / 腾讯云购买）
2️⃣ 先在域名 DNS 管理后台，添加 A 记录，将域名解析指向您服务器的公网 IP
3️⃣ 等待 DNS 生效（通常几分钟，最长可能 24 小时）
4️⃣ DNS 生效后，在下方输入完整域名并点击绑定
⚠️ 注意：必须先完成 DNS 解析配置，再在此处绑定域名，否则绑定后无法访问
📋 IP 绑定说明：
1️⃣ 确认您的服务器有固定公网 IP（可搜索引擎搜索「我的IP」查看）
2️⃣ 确认防火墙已开放对应端口（默认 8000）
3️⃣ 输入公网 IP 并绑定，即可获得固定 HTTP 地址
